Bot Commands

Everything the Unified Events Discord bot can do, in one place. Run a command, follow the prompts.

Slash commands

What you can run

Type either of these directly into any channel the bot can see, or in a DM with the bot.

/forms Opens a DM menu

Sends you a private menu with four options. Reply with a number from 1–4 to choose.

/notify Staff only · Server channels

Sends a notification by DM and posts it to the website's notifications page. Leave the user field empty to message everyone in the server at once.

1 title — Short notification title
2 message — The notification body
3 user — Optional. Pick someone to DM just them
Leave user blank to DM every member in the server instead of one person.
Good to know

A few notes

DMs must be open

The bot reaches you through direct messages, so your server privacy settings need to allow DMs from server members for /forms to work.

One session at a time

You can only have one open ticket, report, or staff chat at once. Close the current one before starting another.

Reports stay anonymized

Staff can respond to a report without seeing your username displayed in the conversation.

You can talk to the bot directly too

Outside of an open session, just message the bot and it'll respond using its built-in knowledge.
