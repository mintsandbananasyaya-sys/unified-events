## What is Unified Events

Unified Events is a structured competitive Minecraft civilization platform where Discord communities compete across seasonal events. Six civilization servers currently participate: Liquids Civilization, Oracles Events, Prite's Events, Crabby's Civilization Experiments, Archie's Civilization Experiments, and Jcrimzy's Events. Discord identity serves as the account foundation for every participant.

## What is Unified Events

Unified Events is a competitive Minecraft civilization event platform. Players form civilizations, build alliances, engage in diplomacy, and compete across structured seasons. It's primarily roleplay and lore-driven — not a PvP-focused server. Events are run on Discord identity and organized into seasons.

## Cant Login

If you're having trouble logging in:
1. Make sure you're a member of the Unified Events Discord server — the login requires server membership.
2. Try clearing your cookies and logging in again.
3. Make sure you're approving the Discord OAuth permissions when prompted.
4. If it still doesn't work, open a ticket via **/forms**.
