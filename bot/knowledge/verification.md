## How to Verify

To verify and unlock the server, run **/setign <your Minecraft username>** in the verification channel. The bot will check your username against Mojang's API. If valid, you'll automatically receive the **Verified** role and gain access to the server. Bedrock players should prefix their username with a dot, e.g. `/setign .YourName`

## Setign

Use **/setign <IGN>** to link your Minecraft account to your Discord. The bot verifies your username against Mojang's API (Java) or accepts it directly for Bedrock (prefix with a dot, e.g. `.Steve`). Once set, you'll get the Verified role and your IGN will appear on the player search page.

<!-- aliases: `setign`, `/setign`, `set ign`, `link minecraft`, `link account`, `minecraft account`, `ign command` -->

## Wrong Ign

If your IGN has changed or you set the wrong one, simply run **/setign <new IGN>** again in the verification channel. It will update your linked account automatically.

## Unsetign

**/unsetign** is a staff-only command used to remove a player's linked IGN and revoke their Verified role. If you need your IGN removed for a legitimate reason, contact staff via **/forms**.

## Java or Bedrock

Both Java and Bedrock players are welcome! Bedrock support is through Geyser/Floodgate. When setting your IGN with **/setign**, Bedrock players should prefix their username with a dot (e.g. `/setign .YourName`). Both editions can apply and participate in Season 1.

## Bot Not Responding

If the bot isn't responding:
1. Make sure you're using slash commands correctly (type `/` and select the command from the menu).
2. Check if the bot has permissions in the channel you're using.
3. Some commands like **/setign** only work in specific channels.
4. If the bot appears offline, it may be restarting â€” try again in a few minutes. If it persists, contact staff via the staff channel.

## Announcements

To get pinged for announcements, grab the **Announcements Ping** role from the roles channel. Other available ping roles include Content Ping, Leaks Ping, and Poll Pings. Check the roles channel in the server to self-assign them.
